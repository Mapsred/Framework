Parses Y\+A\+ML into a P\+HP array.

The parse method, when supplied with a Y\+A\+ML stream (string or file), will do its best to convert Y\+A\+ML in a file into a P\+HP array.

Usage\+: {\ttfamily  \$array = \hyperlink{classSymfony_1_1Component_1_1Yaml_1_1Yaml_ab69c943aac9b8f35804aeccd8a744f24}{Yaml\+::parse}(file\+\_\+get\+\_\+contents(\textquotesingle{}config.\+yml\textquotesingle{})); print\+\_\+r(\$array); }

As this method accepts both plain strings and file names as an input, you must validate the input before calling this method. Passing a file as an input is a deprecated feature and will be removed in 3.\+0.

Note\+: the ability to pass file names to the \hyperlink{classSymfony_1_1Component_1_1Yaml_1_1Yaml_ab69c943aac9b8f35804aeccd8a744f24}{Yaml\+::parse} method is deprecated since version 2.\+2 and will be removed in 3.\+0. Pass the Y\+A\+ML contents of the file instead.
